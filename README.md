## Установка
1. Скопируйте папку giftforyou в `/wa-apps/shop/plugins/`
2. Активируйте плагин в админке: Магазин - Настройки - Плагины
3. Настройте ID товаров в настройках плагина

## Использование
- Страница доступна по URL: /shop/giftforyou/
- Настройте ID товаров через админку (через запятую)

## Отправка писем
**ВАЖНО:** В данной версии плагина реализована **имитация отправки писем** через логирование в файл.

## Как это работает:
1. Пользователь вводит email и нажимает "Отправить"
2. Вместо реальной отправки письма, содержимое письма сохраняется в файл
3. Файл с "отправленными" письмами: `/wa-apps/shop/plugins/giftforyou/logs/sent_emails.log`
